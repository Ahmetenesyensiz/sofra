# Sofra 🍽️

Sofra, restoranlar için geliştirilmiş bir otomasyon ve sipariş yönetim sistemidir. Java Spring Boot ve MongoDB kullanılarak geliştirilmiştir.

## 🚀 Özellikler

- Kullanıcı yönetimi (Admin, Owner, Waiter, Customer)
- Restoran ve masa yönetimi
- Menü ve sipariş yönetimi
- Satış ve stok takibi
- JWT tabanlı kimlik doğrulama ve yetkilendirme

## 🛠️ Teknolojiler

- Java 17
- Spring Boot
- MongoDB
- Spring Security
- JWT
